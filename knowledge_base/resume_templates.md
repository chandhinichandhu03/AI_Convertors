# Resume Templates & Structure Reference

A guide outlining standard resume sections, ordering, and structural templates.

## 1. Structural Layout Templates

### Chronological Resume Template
Best for candidates with a steady, linear career progression in the same industry.
* Contact Info
* Professional Summary
* Work Experience (Reverse-chronological order: present to past)
* Projects (Optional or integrated)
* Education
* Skills

### Functional / Skills-Based Template
Best for career changers, freelancers, or candidates with employment gaps.
* Contact Info
* Professional Summary
* Core Skills Summary (Detailed list of skills grouped by category with brief proof/examples)
* Work History (List of job titles, companies, and dates only - without bullet points)
* Education

### Hybrid / Combination Template
Best for showcasing deep technical skills alongside professional growth.
* Contact Info
* Professional Summary
* Key Skills
* Professional Experience (Bullet points focusing on achievements)
* Key Projects (Focusing on specific technical challenges solved)
* Education

## 2. Standard Bullet Formulations
* **Format**: `[Action Verb] + [Task / Project Description] + [Method / Tool Used] + [Quantifiable Metric / Business Result]`
* **Example (Frontend)**: "Engineered a client-side state caching layer using React Context API, reducing network request latency by 45% and saving 2.5s of page load time."
* **Example (Backend)**: "Refactored PostgreSQL database schemas and optimized indexing strategies, reducing API response times for transactional endpoints by 60%."
