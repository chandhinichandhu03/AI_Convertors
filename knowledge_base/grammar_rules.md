# Grammar and Writing Style Rules

This guide outlines principles of clear, impactful writing for professional resumes, cover letters, and technical documents.

## 1. Action-Oriented Tone (Active Voice)
* Write bullet points in the active voice. Passive voice sounds weak and takes the credit away from you.
* *Passive*: "A database performance issue was resolved by me."
* *Active*: "Resolved a critical database performance bottleneck, improving query execution speed by 40%."

## 2. Strong Professional Action Verbs
Use specific action verbs. Avoid repetitive terms like "Managed" or "Responsible for".
* **For Technical Achievements**: "Engineered", "Developed", "Refactored", "Architected", "Optimized", "Integrated", "Programmed", "Deployed".
* **For Leadership & Innovation**: "Spearheaded", "Directed", "Coached", "Pioneered", "Orchestrated", "Implemented", "Established", "Mentored".
* **For Efficiency & Savings**: "Streamlined", "Accelerated", "Consolidated", "Minimized", "Yielded", "Reduced", "Maximized", "Trimmed".

## 3. Pronoun Usage
* Resumes should omit first-person pronouns (e.g., "I", "me", "my", "we"). Instead of writing "I developed the software," start directly with the verb: "Developed the software."

## 4. Consistent Tense Alignment
* Match the verb tense to the employment status:
  * **Current Job Experience**: Write bullets in the **present tense** (e.g., "Develop applications", "Collaborate with cross-functional teams").
  * **Past Job Experience**: Write bullets in the **past tense** (e.g., "Developed application", "Collaborated with team").
* Ensure that within a single job listing, tenses are aligned and do not alternate between past and present.

## 5. Parallelism
* Maintain parallel structure in lists and bullet points. If the first bullet starts with an action verb, all subsequent bullets in that section must start with an action verb.
