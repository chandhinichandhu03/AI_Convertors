# Job Description Samples

A reference list of standard job descriptions across major technology tracks to help the AI map resume gaps and optimize skills coverage.

## 1. Frontend Developer (React/TypeScript)
* **Overview**: Seeking a Frontend Developer to build responsive, high-performance web applications.
* **Key Responsibilities**:
  * Develop modular web components using React, TypeScript, and Tailwind CSS.
  * Optimize rendering pipelines, lazy loading, and state management.
  * Integrate RESTful and GraphQL APIs.
* **Requirements**:
  * 3+ years experience with modern React, TypeScript, and Git.
  * Deep understanding of HTML5 semantic structure, modern CSS grid/flexbox, and client-side performance.
  * Experience with unit testing (Jest/Vitest).

## 2. Backend Engineer (Python/FastAPI/SQL)
* **Overview**: Looking for a Backend Engineer to design scalable APIs and manage database operations.
* **Key Responsibilities**:
  * Build REST APIs using Python, FastAPI, and SQLAlchemy.
  * Optimize database schemas, queries, and migrations in PostgreSQL / SQLite.
  * Implement authentication systems (JWT, OAuth2) and secure endpoints.
* **Requirements**:
  * Proficiency in Python and API design.
  * Experience with SQL, relational database indexing, and query tuning.
  * Understanding of Docker, CI/CD pipelines, and cloud hosting.

## 3. AI / Machine Learning Engineer
* **Overview**: Seeking an AI Engineer to develop and deploy machine learning models and RAG search pipelines.
* **Key Responsibilities**:
  * Implement local vector databases (FAISS, Chroma) and embed documents using sentence-transformers.
  * Integrate Large Language Models (LLMs) like Llama, DeepSeek, or Gemma.
  * Design chunking, parsing, and pipeline orchestration (LangChain/LlamaIndex).
* **Requirements**:
  * Core knowledge of Python, NumPy, Pandas, Scikit-learn, and PyTorch.
  * Experience building RAG pipelines, prompt engineering, and fine-tuning.
  * Understanding of vector math, embeddings, and similarity metrics.

## 4. DevOps Engineer
* **Overview**: Seeking a DevOps Engineer to automate build, deployment, and infrastructure orchestration.
* **Requirements**: Docker, Kubernetes, Terraform, AWS/GCP, GitHub Actions, Linux administration.

## 5. Cybersecurity Specialist
* **Overview**: Seeking a security specialist to monitor, test, and patch network infrastructure and application endpoints.
* **Requirements**: OWASP Top 10, penetration testing, firewalls, threat modeling, network protocols, ISO 27001.

## 6. Data Scientist
* **Overview**: Seeking a Data Scientist to parse, analyze, and build predictive statistical models from large data matrices.
* **Requirements**: Python, R, SQL, Pandas, Jupyter, statistical modeling, linear regression, clustering, Tableau.
