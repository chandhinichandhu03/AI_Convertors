# Mathematics, Computer Science & Cyber Security Reference

This reference covers calculus formulas, CS architectures, networking layers, cryptography, and machine learning logic.

## 1. Mathematical Formulas
* **Derivatives**:
  * $\frac{d}{dx}(x^n) = n x^{n-1}$
  * $\frac{d}{dx}(\sin x) = \cos x$
  * $\frac{d}{dx}(e^x) = e^x$
* **Integrals**:
  * $\int x^n dx = \frac{x^{n+1}}{n+1} + C$
  * $\int \frac{1}{x} dx = \ln|x| + C$
* **Linear Algebra**: Matrix transpose $A^T$, matrix inverse $A^{-1}$, determinants, and eigenvectors.
* **Boolean Algebra**: De Morgan's Laws: $\neg(A \land B) \iff \neg A \lor \neg B$ and $\neg(A \lor B) \iff \neg A \land \neg B$.

## 2. Computer Science & Graph Algorithms
* **Binary Trees**: Hierarchical structures where nodes have at most two children (left and right). Binary Search Trees (BST) maintain sorted order: left child < parent < right child.
* **Fourier Transforms (FFT)**: Fast Fourier Transform (FFT) computes the Discrete Fourier Transform (DFT) in $O(N \log N)$ complexity rather than $O(N^2)$, enabling fast digital signal processing.
* **Graph Traversals**: Breadth-First Search (BFS) uses a queue for level-order search. Depth-First Search (DFS) uses a stack or recursion for path-finding.

## 3. Networking & Cyber Security
* **OSI 7-Layer Model**: Physical, Data Link, Network (IP), Transport (TCP/UDP), Session, Presentation, Application (HTTP/SMTP).
* **Cryptography**: Symmetric (AES, DES) vs Asymmetric (RSA, ECC) encryption. RSA uses prime factorization as a trapdoor function.
* **Cyber Security**: Common vectors: SQL Injection (SQLi), Cross-Site Scripting (XSS), CSRF, MITM, DDoS.

## 4. Machine Learning & Artificial Intelligence
* **Supervised Learning**: Linear Regression, Support Vector Machines (SVM), Random Forest, Neural Networks.
* **Unsupervised Learning**: K-Means clustering, Principal Component Analysis (PCA) for dimension reduction.
* **Natural Language Processing (NLP)**: Tokenization, embeddings (e.g. Word2Vec), Transformer self-attention mechanism.
